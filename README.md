## 安装

```bash
# 最新稳定版
$ npm install vue@next

# 命令行工具
$ npm install -g @vue/cli
vue upgrade --next

# vite 快速构建vue项目
$ npm init vite-app <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```
